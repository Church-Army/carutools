# See all Church Army palette and colour names

A character vector of colour or pallette names accepted by `carutools`
functions. For instance,
[ca_pal](https://church-army.github.io/carutools/reference/ca_pal.md)
will only accept a name from `ca_sample_pals()`.

## Usage

``` r
ca_sample_pals()

ca_sample_cols()
```

## Value

A character vector of Church Army palette or colour names
