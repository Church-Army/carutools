# Church Army Colours

These functions each return the hexadecimal code of a colour in Church
Army's brand

## Usage

``` r
ca_black()

ca_white()

ca_orange()

ca_gold()

ca_green()

ca_maroon()

ca_cyan()

ca_purple()

ca_light_teal()

ca_dark_teal()

ca_brown()
```

## Value

A single character vector of length one.

## Details

These functions are wrappers around character vectors of length one.
They take no arguments and always return the same string.

## Examples

``` r
ca_black()
#> [1] "#000000"
```
