# Church Army Colour Pallets

These functions each return a colour pallet from Church Army's brand

## Usage

``` r
ca_pal_orange()

ca_pal_gold()

ca_pal_light_teal()

ca_pal_cyan()

ca_pal_dark_teal()

ca_pal_maroon()

ca_pal_purple()

ca_pal_brown()

ca_pal_green()

ca_pal_mix()
```

## Value

A single character vector of length five (nine in the case of
`ca_pal_mix()`), containing colour hexcodes.

## Details

These functions are wrappers around character vectors of length five.
They take no arguments and always return the same character vector.
